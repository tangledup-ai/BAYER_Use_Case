# 拜耳制药排班系统 - Windows打包指南

本文档说明如何将FastAPI后端应用正确打包为Windows可执行文件（.exe）。

## 前置要求

### 1. Python环境
- Python 3.8 或更高版本
- 推荐使用Python 3.9+ 以获得更好的兼容性

### 2. 安装依赖

首先安装项目依赖：

```bash
# 安装FastAPI及相关依赖
pip install -r requirements.txt

# 或单独安装
pip install fastapi uvicorn pydantic pyinstaller
```

## 打包步骤

### 方法一：使用打包脚本（推荐）

1. **双击运行打包脚本**

   双击 `build.bat` 文件，系统将自动执行以下操作：
   - 检查Python环境
   - 安装PyInstaller（如未安装）
   - 安装项目依赖
   - 清理旧构建文件
   - 执行打包操作
   - 生成可执行文件

2. **等待打包完成**

   打包过程可能需要几分钟时间，取决于项目大小和依赖数量。

3. **运行测试**

   打包成功后，脚本会询问是否立即运行测试。选择"Y"可以测试打包后的程序。

### 方法二：手动打包

如果你更喜欢手动操作，可以按以下步骤进行：

1. **安装依赖**

```bash
pip install -r requirements.txt
```

2. **执行打包命令**

```bash
pyinstaller build.spec --clean
```

3. **等待打包完成**

   打包后的文件将位于 `dist\拜耳排班系统\` 目录下。

## 打包输出

打包成功后，可执行文件位于：

```
dist\拜耳排班系统\拜耳排班系统.exe
```

打包后的目录结构：

```
dist\拜耳排班系统\
├── 拜耳排班系统.exe          # 主程序
├── backend\                   # 后端代码
│   ├── app.py
│   └── requirements.txt
├── frontend\                  # 前端文件
│   ├── index.html
│   ├── css/
│   ├── js/
│   └── pages/
└── [Python运行库和依赖]
```

## 运行打包后的程序

1. **直接运行**

   双击 `拜耳排班系统.exe` 文件

2. **程序将自动**：
   - 启动Web服务器（端口5000）
   - 打开浏览器访问应用
   - 显示启动信息

3. **访问地址**：
   - 应用主页：http://127.0.0.1:5000/
   - API文档：http://127.0.0.1:5000/docs

## 常见问题

### 问题1：打包失败，提示找不到模块

**解决方案**：
- 确保所有依赖都已安装：`pip install -r requirements.txt`
- 检查是否有自定义模块未包含在打包配置中
- 查看打包日志中的具体错误信息

### 问题2：程序启动后立即退出

**解决方案**：
- 检查 `frontend` 文件夹是否与 exe 文件在同一目录
- 查看是否有端口冲突（5000端口被占用）
- 运行打包后的程序时查看控制台输出（临时将console设为True）

### 问题3：程序启动正常但无法加载页面

**解决方案**：
- 确保 `frontend` 文件夹完整存在
- 检查浏览器控制台是否有错误信息
- 尝试使用Chrome或Firefox等现代浏览器

### 问题4：打包文件过大

**解决方案**：
- 使用虚拟环境隔离项目依赖
- 排除不必要的模块（在build.spec的excludes中配置）
- 考虑使用UPX压缩（需要安装UPX工具）

## 高级配置

### 修改打包配置

编辑 `build.spec` 文件可以自定义打包行为：

```python
# 修改可执行文件名
name='你的应用名称'

# 修改控制台模式（True显示控制台窗口）
console=False

# 修改UPX压缩设置
upx=False
```

### 添加自定义图标

1. 准备一个 `.ico` 格式的图标文件
2. 在 `build.spec` 中添加：

```python
exe = EXE(
    # ... 其他配置 ...
    icon='你的图标.ico',
)
```

### 修改服务器端口

如需修改默认端口（5000），编辑以下文件：
- `backend/app.py` 中的 `uvicorn.run()` 调用
- `run_app.py` 中的 `server_url` 变量

## 卸载

删除 `dist\拜耳排班系统\` 整个文件夹即可。

## 技术支持

如果遇到其他问题，请检查：
1. Windows事件查看器中的应用日志
2. 打包过程中的错误输出信息
3. 程序的端口占用情况（netstat -ano | findstr :5000）
